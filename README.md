# qutebrowser Configuration Notes

## General Settings
- Loads settings from `autoconfig.yml`.
- Start page set to Google.

## Search Engines
- Custom search engines configured:
  - `DEFAULT`: Google
  - `ddg`: DuckDuckGo
  - `b`: Bing
  - `yt`: YouTube
  - `gh`: GitHub
  - `wp`: Wikipedia
  - `in`: Instagram
  - `li`: LinkedIn
  - `an`: HiAnime

## Privacy & Security
- Ad blocking enabled using EasyList and EasyPrivacy.
- JavaScript is enabled.
- Custom user agent set for better compatibility.

## UI Customization
- Default font size: 12pt.
- Dark mode for webpages is enabled.
- Webpage background color: `#1e1e1e`.

## Theme Customization
### Completion Widget
- Background: Dark (`#222222`)
- Selected item: Highlighted (`#555555` with yellow match `#ffcc00`)

### Status Bar
- Normal mode: Dark (`#222222`)
- Insert mode: Blue (`#005f87`)
- Passthrough mode: Purple (`#8700af`)
- Command mode: Dark (`#333333`)

### Tab Bar
- Background: Dark (`#111111`)
- Selected tab: Highlighted (`#444444` and `#555555`)
- Text: White (`#ffffff`)

## Key Bindings
- `1` → Switch to previous tab
- `2` → Switch to next tab
- `Alt + Left` → Go back in history
